# Nanuri
프로젝트

###개요
> 렌탈 서비스를 제공하는 가상의 사업체를 위한 웹사이트(소셜커머스)를 제작한다.

> 프로젝트 인원 : 5명
> 나의 기여도   : 35%
> 담당역활      : 렌탈게시판 , 데이터베이스 설계 , 디자인


###개발환경
● 언어
    JAVA 8 { JSP , MVC2  }
● 프레임워크
    Spring { Mybatis , tiles }
● DB
   ORACLE 11g

● HTML / CSS / JavaScript
   JQuery / bootstrap
